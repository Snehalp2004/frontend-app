# HireHelper - Frontend User Guide

## What is HireHelper?
HireHelper is a full-stack web application designed to connect users who need help with tasks (hirers) and those willing to offer services (helpers). Key features include:
- User authentication (login/register/forgot password/OTP verification)
- Profile management with photo uploads
- Dashboard for overview
- Posting and managing tasks/requests
- Viewing feeds, my tasks, my requests, offers
- Notifications
- Help section

Built with Angular frontend.

## User Guide
1. **Getting Started**:
   - Visit the landing page.
   - Register a new account or login.

2. **Profile Setup**:
   - Navigate to profile page to upload photo and edit details.

3. **Posting Tasks/Requests**:
   - Use 'Add Task' or 'Requests' page to create new tasks/requests.

4. **Managing Content**:
   - Dashboard: Overview of your activities.
   - My Tasks: View/edit your posted tasks.
   - My Requests: Manage requests.
   - Feed: Browse available tasks.
   - Notifications: Stay updated.

5. **Collaboration**:
   - Make offers on requests.
   - Respond to notifications.

For developers: Run `ng serve` in frontend-app/frontend/ to start the app at http://localhost:4200.
